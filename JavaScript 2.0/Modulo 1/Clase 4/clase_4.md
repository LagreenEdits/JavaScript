## Temas que probablemente se cubrieron (usa esto como mínimo, amplía si falta algo lógico)
- Estructura base de un documento HTML: `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`
- Etiquetas de texto: `<h1>`-`<h6>`, `<p>`, `<span>`, `<strong>`, `<em>`
- Etiquetas de estructura semántica: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`
- Listas: `<ul>`, `<ol>`, `<li>`
- Enlaces e imágenes: `<a href>`, `<img src alt>`
- Formularios básicos: `<form>`, `<input>`, `<label>`, `<button>`, tipos de input
- Atributos globales: `class`, `id`, `style`
- Introducción a CSS: cómo vincularlo (inline, interno `<style>`, externo `<link>`)
- Selectores CSS: por etiqueta, por clase (`.`), por id (`#`), selector universal (`*`)
- Especificidad básica de selectores (qué gana si hay conflicto)
- Propiedades de texto: `color`, `font-size`, `font-family`, `text-align`
- Propiedades de caja (box model): `margin`, `padding`, `border`, `width`, `height`
- Modelo de caja: `content-box` vs `box-sizing: border-box`
- Unidades de medida: `px`, `%`, `em`, `rem`
- Colores: nombres, hexadecimal, `rgb()`, `rgba()`

## Formato de salida requerido (usa EXACTAMENTE esta estructura)

Para el tema completo de la clase, entrega:

1. **Índice de subtemas** cubiertos en la clase de 3h (lista completa y ordenada)
2. **Explicación detallada por subtema** (no solo definición — explica el "por qué" y "cómo funciona")
3. **Conceptos clave** (glosario con término + definición corta)
4. **¿Para qué sirve cada concepto?** en la práctica real
5. **Ejemplos abstractos** usando notación `[nombre_1]`, `[valor_1]`, indicando cuáles son:
   - Reutilizables/renombrables por el estudiante (nombres de clases, ids)
   - Fijos del lenguaje (etiquetas HTML, propiedades CSS — no se inventan nombres nuevos)
6. **Excepciones y casos especiales** (id único vs class repetible, especificidad CSS, herencia de estilos, cuándo `box-sizing` cambia el comportamiento)
7. **Metáfora o ejemplo visual** para cada concepto difícil de imaginar
8. **Errores típicos** que comete un junior en este tema (olvidar cerrar etiquetas, confundir class con id, no vincular bien el CSS, no entender el box model)
9. **Preguntas de autoevaluación** (mínimo 8) tipo "¿Puedo explicar...?" para que yo verifique si realmente entendí
10. **Checklist final** de dominio del tema (formato casillas ✅)
